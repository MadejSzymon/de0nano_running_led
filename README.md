# de0nano_running_led
Running LED done on DE0-Nano FPGA Development Kit

SOF file is ready for upload to FPGA.


KEY0: Reset

KEY1: Change direction

SW[3:0]: Change speed

  Speed table: (numbers on the left side represent SW inputs [SW3,SW2,SW1,SW0])
  
    1xxx : 8 Hz
    
    01xx : 4 Hz
    
    001x : 2 Hz
    
    0001 : 1 Hz
    
    0000 : 0.5 Hz
